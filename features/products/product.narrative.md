# Products Search & Add to Cart Feature - Acceptance Criteria

### **Feature Description**:
Users can search for products and add available items to their cart.

### **Acceptance Criteria:**

1. **Products Page Load**
   - The page title should display as "Product List MFE".

2. **Product Search**
   - When a search term is entered, the total product count should match the search results.

3. **Add to Cart**
   - Users can add "In Stock" products to the cart.
   - Users cannot add "Out of Stock" products to the cart and will see add to cart button disabled.
